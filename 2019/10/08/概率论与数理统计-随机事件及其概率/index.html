概率论与数理统计-随机事件及其概率

2019-10-08

样本空间、随机事件概念，事件关系及运算

随机事件

  1. 随机试验 EEE 的特征：
    1. 在相同条件下，实验可重复执行。
    2. 实验的所有可能结果事先已知。
    3. 实验的每次具体结果事先未知。
  2. 样本点 ωωω：随机试验的每一个基本可能结果。
  3. 样本空间 ΩΩΩ：随机试验的所有样本点。
  4. 随机事件：样本空间的子集，常用大写字母 A,B,C,⋯A, B, C, \cdotsA,B,C,⋯ 表示事件。
  5. 必然事件 ΩΩΩ：每次实验中必然发生的事件，全集。
  6. 不可能事件 ∅\empty∅： 每次实验都不可能发生的事件，空集。

事件的关系与运算

  1. 事件的三种关系与四种运算
    1. 子事件 A⊂BA \subset BA⊂B：AAA 发生，BBB 也发生。
    2. 相等事件 A=BA = BA=B：A⊂BA \subset BA⊂B 且 A⊃BA \supset BA⊃B。
    3. 并（和）事件 A∪BA \cup BA∪B：至少有一个发生。
    4. 交（积）事件 A∩BA \cap BA∩B：都发生。
    5. 对立事件：A∪B=ΩA \cup B = \OmegaA∪B=Ω 且 AB=∅AB = \emptyAB=∅，对于事件 A,B,A‾=BA, B, \overline{A} = BA,B,A=B
    6. 差事件 A−BA - BA−B：AAA 发生，BBB 不发生
    7. 互斥事件：AB=∅AB = \emptyAB=∅
  2. 事件的运算律：
    1. 交换律：A∪B=B∪A,AB=BAA \cup B = B \cup A, AB = BAA∪B=B∪A,AB=BA
    2. 结合律：(A∪B)∪C=A∪(B∪C),(AB)C=A(BC)(A \cup B) \cup C = A \cup (B \cup C), (AB)C = A(BC)(A∪B)∪C=A∪(B∪C),(AB)C=A(BC)
    3. 吸收律：若 A⊂BA \subset BA⊂B，则 AB=A,A∪B=BAB = A, A \cup B = BAB=A,A∪B=B
    4. 分配律：(A∪B)∩C=(A∩C)∪(B∩C)(A \cup B) \cap C = (A \cap C) \cup(B \cap C)(A∪B)∩C=(A∩C)∪(B∩C)
    5. 对偶律：A∪B‾=A‾⋅B‾,AB‾=A‾∪B‾\overline{A \cup B} = \overline{A}\cdot\overline{B}, \overline{AB} = \overline{A} \cup \overline{B}A∪B=A⋅B,AB=A∪B

概率、条件概率

概率的概念

直观上讲，是指一个事件发生的可能性大小的数量指标。

  1. 概率的统计定义：条件不变的情况下，重复进行 nnn 次实验，事件 AAA 发生的频率稳定的在一常数 ppp 附近摆动，且一般来说，nnn 越大，摆动越小，则称常数 ppp 为事件 AAA 的概率，记作 P(A)=pP(A) = pP(A)=p。
  2. 概率的公理化定义：设随机试验 EEE 的样本空间为 Ω\OmegaΩ 对于 EEE 的每个事件 AAA 赋予一实数 P(A)P(A)P(A)，若 P(⋅)P(\cdot)P(⋅) 满足以下条件：
    1. 非负性：对于每一个事件 AAA 都有 P(A)⩾0P(A) \geqslant 0P(A)⩾0
    2. 规范性：对于必然事件 P(Ω)=1P(\Omega) = 1P(Ω)=1
    3. 可列可加性：若 A1,A2,⋯ ,An,⋯A_1,A_2,\cdots,A_n,\cdotsA1​,A2​,⋯,An​,⋯是两两互不相容的事件，则有 P(∪i=1∞Ai)=∑i=1∞P(Ai) P(\displaystyle\cup_{i=1}^\infty A_i) = \displaystyle\sum_{i=1}^\infty P(A_i)P(∪i=1∞​Ai​)=i=1∑∞​P(Ai​) 就称 P(A)P(A)P(A) 为事件 AAA 的概率
  3. 概率的性质：
    1. p(∅)=0,p(Ω)=1p(\empty) = 0, p(\Omega) = 1p(∅)=0,p(Ω)=1
    2. 若 A1,A2,⋯ ,An,⋯A_1,A_2,\cdots,A_n,\cdotsA1​,A2​,⋯,An​,⋯是两两互不相容的事件则有 P(∪i=1∞Ai)=∑i=1∞P(Ai) P(\displaystyle\cup_{i=1}^\infty A_i) = \displaystyle\sum_{i=1}^\infty P(A_i)P(∪i=1∞​Ai​)=i=1∑∞​P(Ai​)
    3. 求逆公式：P(A‾)=1−P(A)P(\overline{A}) = 1 - P(A)P(A)=1−P(A)
    4. 减法公式：P(A−B)=P(AB‾)=P(A)−P(AB)P(A-B) = P(A\overline{B}) = P(A) - P(AB)P(A−B)=P(AB)=P(A)−P(AB)，特别地，当 B⊂AB \subset AB⊂A 时， P(A−B)=P(A)−P(B)P(A-B) = P(A) - P(B)P(A−B)=P(A)−P(B)，由此可得 P(B)⩽P(A)P(B) \leqslant P(A)P(B)⩽P(A)
    5. 加法公式：P(A∪B)=P(A)+P(B)−P(AB)P(A \cup B) = P(A) + P(B) - P(AB)P(A∪B)=P(A)+P(B)−P(AB)，P(A∪B∪C)=P(A)+P(B)+P(C)−P(AB)−P(BC)−P(AC)+P(ABC)P(A \cup B \cup C) = P(A) + P(B) + P(C) - P(AB) - P(BC) - P(AC) + P(ABC)P(A∪B∪C)=P(A)+P(B)+P(C)−P(AB)−P(BC)−P(AC)+P(ABC)
      特别地，P(A1∪A2∪⋯∪An)=P(A1)+P(A2)+⋯+P(An)P(A_1 \cup A_2 \cup \cdots \cup A_n) = P(A_1) + P(A_2) + \cdots + P(A_n)P(A1​∪A2​∪⋯∪An​)=P(A1​)+P(A2​)+⋯+P(An​)

古典概型和几何概型

  1. 古典概型：
    1. 特征：Ω\OmegaΩ 中的样本点有限，且每个样本点发生的可能性相同
    2. 公式：P(A)=mnP(A) = \frac{m}{n}P(A)=nm​，其中 mmm 为 A 包含的基本事件数，nnn 为 Ω\OmegaΩ 中基本事件的总数。
      常用结论：设 NNN 件产品中，有 MMM 件次品，从中任取 nnn 件，则其中恰有 kkk 件次品的概率为 CMkCN−Mn−k，其中CNn=N(N−1)⋯[N−(n−1)]n! \frac{C_M^k}{C_{N-M}^{n-k}}，其中 C_N^n = \frac{N(N - 1)\cdots[N - (n - 1)]}{n!} CN−Mn−k​CMk​​，其中CNn​=n!N(N−1)⋯[N−(n−1)]​
  2. 几何概型：
    1. 特征：Ω\OmegaΩ 中的样本点无限且构成一个几何区域，且每个样本点发生的可能性相同
    2. 公式：P(A)=L(A)L(Ω)P(A) = \frac{L(A)}{L(\Omega)}P(A)=L(Ω)L(A)​，L(A)L(A)L(A) 和 L(Ω)L(\Omega)L(Ω) 分别表示 AAA 和 Ω\OmegaΩ 的几何测度，其中测度为长度和面积。
