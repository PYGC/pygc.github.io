Vim 备忘单(快捷键)

2019-10-05

导航

常规

按键 含义
h 左移
j 下移
k 上移
l 右移
10j 下移 10 行

左、右、上、下箭头键也可以用来导航。

当前屏幕 (Screen)

按键 含义
H 屏幕顶行
M 屏幕中间行
L 屏幕底行

文件 (File)

按键 含义
gg 文件第一行
G 文件最后一行
:20 文件第 20 行

字 (Word)

按键 含义
e 当前词的结束
b 当前词的开始
w 下一个词的开始

行 (Line)

按键 含义
0 当前行的开始
^ 当前行的首个非空白字符
$ 当前行的结束
% 移至匹配的括号，方括号或大括号

编辑

按键 含义
i 在当前字符之前插入
a 在当前字符之后插入
I 在该行的第一个非空白字符处插入
o 在当前行下方插入一行，然后进入插入模式
O 在当前行上方插入一行，然后进入插入模式
r 覆盖一个字符并返回命令模式
U 撤消
Ctrl+R 重做

打开，关闭和保存文件

按键 含义
:w 保存当前文件
:wq 保存当前文件并关闭它； 如果没有剩余打开的文件，退出vim
:w newname 将当前文件的副本另存为 "newname"，但继续编辑原始文件
:sav newname 将当前文件的副本另存为 " newname"，然后继续编辑文件
:q! 关闭文件而不保存
:e somefile 在当前缓冲区中打开文件
:x 将所有更改写入文件并关闭文件

模式切换

按键 含义
i 进入插入模式
: 进入命令模式
R 进入替换模式
v 进入可视模式 (突出显示)
V 进入可视行模式 (高亮显示行)
esc 从插入或替换模式返回正常模式
esc+esc 从命令或可视模式返回正常模式

复制/粘贴

在 vim 内

按键 含义
y 拖动/复制所选区域
c "change"; 剪切并进入插入模式
C 更改当前行的其余部分
d "delete"; 剪切但保持在正常模式
D 删除当前行的其余部分
p 光标后粘贴
P 光标前粘贴
x 删除光标后的字符
X 删除光标前的字符

复制/粘贴命令在指定范围内运行。 如果处于可视模式，则该范围是突出显示的文本。 如果在普通模式下，则该范围由命令的一系列修饰符指定：

按键 含义
cw 更改一个字(Word)
c4w 更改四个字
c4l 更改四个字母
cc 更改当前行
4x 更改光标后四个字符
5p 在光标后粘贴五次

修改器对剪切、删除、拉拽和粘贴的作用类似。

从系统剪贴板

按键 含义
:set paste 进入粘贴模式
:set nopaste 退出粘贴模式
Ctrl+Shift+V 粘贴到文件 (如果处于粘贴模式)； Command+Shift+V 对于Mac

替换

按键 含义
:s/foo/bar/ 用 bar 代替当前行的第一个 foo
:[range]s/foo/bar/[flags] 在范围 (range) 内根据标志 (flag) 用 bar 替换 foo

范围

按键 含义
% 整个文件
'<,'> 当前选择； 可视模式下的默认范围
25 第 25 行
25,50 第 25 到 50 行
$ 最后一行； 可以与其他行组合 50,$
. 当前行; 可以与其他行组合 .,50
,+2 当前行和下面的两行
-2, 当前行及其上方的两行

标志

按键 含义
g 替换指定行上的所有匹配项
i 忽略大小写
c 确认每次替换

正则表达式

vim 的查找和替换功能均接受正则表达式。 默认情况下，vim 假定某些字符是正则表达式的一部分，并且这些字符必须用 \ 进行转义才能进行字面搜索； 这些字符包括 (, ), *, ., ^, $。 其他正则表达式模式默认情况下会按字面解释，并且必须转义以用作正则表达式的一部分。 这些包括 +。

搜索

按键 含义
* 查找当前单词的下一个实例
# 查找当前单词的上一个实例
n 按照上次使用 *,＃ 所指定的方向，找到要搜索的单词的下一个实例
N 按照上次使用 *,＃ 所指定的方向找到要搜索的单词的上一个实例
/word 查找下一个出现的 word
/word\c 查找下一个出现的 word，忽略大小写（ \c 可以出现在要搜索的顺序中的任何位置）
/\<word\> 查找下一个出现的单词 word，其中 word 以单词边界为界（例如空格，破折号）
:noh 不突出显示的单词

有用的技巧

按键 含义
- 切换光标下的字符大小写
xp 将当前字符移到右边
J 连接当前行及其下方的行
:Ni! 彩蛋
:help 查看 vim 的帮助文件
:help foo 查看 vim 帮助的 foo 条目
