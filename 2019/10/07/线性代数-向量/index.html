向量组的相关性与线性表示

基本概念

  1. 向量的线性组合:
    k1α1+k2α2+⋯+knαnk_1\alpha_1 + k_2\alpha_2 + \cdots + k_n\alpha_nk1​α1​+k2​α2​+⋯+kn​αn​ 为向量组 α1,α2,⋯+αn\alpha_1, \alpha_2, \cdots + \alpha_nα1​,α2​,⋯+αn​ 的线性组合。
  2. 向量的相关性:
    齐次线性方程组：x1α1+x2α2+⋯+xnαn=0x_1\alpha_1 + x_2\alpha_2 + \cdots + x_n\alpha_n = 0x1​α1​+x2​α2​+⋯+xn​αn​=0
    只有零解, 称向量组 α\alphaα 线性无关。
    有非零解，称向量组 α\alphaα 线性相关。
  3. 向量的线性表示：
    非齐次线性方程组：x1α1+x2α2+⋯+xnαn=bx_1\alpha_1 + x_2\alpha_2 + \cdots + x_n\alpha_n = bx1​α1​+x2​α2​+⋯+xn​αn​=b
    有解, 称向量 bbb 可由 α\alphaα 线性表示。
    无解, 称向量 bbb 不可由 α\alphaα 线性表示。

向量组相关性与线性表示的性质

  1. 向量组线性相关，充分必要条件是，至少有一个向量可有其余向量线性表示。
  2. α\alphaα 向量线性无关：
    1. α,b\alpha, bα,b 线性相关，则 bbb 向量可由 α\alphaα 唯一线性表示。
    2. α,b\alpha, bα,b 线性无关，则 bbb 向量不可由 α\alphaα 线性表示。
  3. 若一个向量组线性无关，则该向量组的任何部分都线性无关
  4. 若向量组的部分线性相关，则该向量组一定线性相关

向量组等价、向量组的极大线性无关组与向量组的秩

基本概念

  1. 向量组等价：
    α,β\alpha, \betaα,β 可互相线性表示，则 向量组α,β\alpha, \betaα,β等价。
  2. 向量组的极大线性无关组与向量组的秩

向量组秩的性质

  1. 矩阵的行向量组与列向量组的秩相等
  2. AAA向量组可由BBB向量组线性表示，则AAA向量组的秩不超过BBB向量组的秩。
  3. 等价的向量组的秩相等，反之不对。

  1. nnn 维向量空间
  2. 基与正交规范基
  3. 向量在基下的坐标
  4. 过度矩阵

基本性质

  1. 向量 β\betaβ 在基 α\alphaα 下的坐标为 X=A−1βX = A^{-1}\betaX=A−1β
  2. 过度矩阵Q=A−1BQ = A^{-1}BQ=A−1B
  3. α\alphaα 到 β\betaβ 的过度矩阵与 β\betaβ 到 α\alphaα 的过度矩阵互为逆矩阵。
