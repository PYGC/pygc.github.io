线性方程组解的基本定理

  1. AAA 为 m∗nm * nm∗n 矩阵，
    1. 齐次线性方程组 AX=0AX = 0AX=0 只有零解的充要条件是 r(A)=nr(A) = nr(A)=n.
    2. 齐次线性方程组 AX=0AX = 0AX=0 有非零解的充要条件是 r(A)<nr(A) < nr(A)<n.
  2. AAA 为 nnn 阶矩阵，
    1. 齐次线性方程组 AX=0AX = 0AX=0 只有零解的充要条件是 ∣A∣≠0|A| \neq 0∣A∣​=0.
    2. 齐次线性方程组 AX=0AX = 0AX=0 有非零解的充要条件是 ∣A∣=0|A| = 0∣A∣=0.
  3. AAA 为 m∗nm * nm∗n 矩阵，增广矩阵 Aˉ=(A⋮b)\bar{A} = (A \vdots b)Aˉ=(A⋮b)，
    1. 非齐次线性方程组 AX=bAX = bAX=b 有解的充要条件是 r(A)=r(Aˉ)r(A) = r(\bar{A})r(A)=r(Aˉ),
      1. r(A)=r(Aˉ)=nr(A) = r(\bar{A}) = nr(A)=r(Aˉ)=n 时有唯一解.
      2. r(A)=r(Aˉ)<nr(A) = r(\bar{A}) < nr(A)=r(Aˉ)<n 时有无数个解.
    2. 非齐次线性方程组 AX=bAX = bAX=b 无解的充要条件是 r(A)≠r(Aˉ)r(A) \neq r(\bar{A})r(A)​=r(Aˉ).
  4. AAA 为 nnn 阶矩阵，
    1. 非齐次线性方程组 AX=bAX = bAX=b 有解的充要条件是 r(A)=r(Aˉ)r(A) = r(\bar{A})r(A)=r(Aˉ),
      1. ∣A∣≠0|A| \neq 0∣A∣​=0 时有唯一解.
      2. ∣A∣=0|A| = 0∣A∣=0 时有无数个解.
    2. 非齐次线性方程组 AX=bAX = bAX=b 无解的充要条件是 r(A)≠r(Aˉ)r(A) \neq r(\bar{A})r(A)​=r(Aˉ).
