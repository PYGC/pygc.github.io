USB的几种速度

2019-10-31

截至 USB 3.1 时，USB 支持五种总线速度：

  • Low speed: 1.5 Mbps
  • Full speed: 12 Mbps
  • High speed: 480 Mbps
  • SuperSpeed: 5 Gbps
  • SuperSpeedPlus: 10 Gbps

在 PC 上: - USB 3.1 Host 控制器支持所有五种速度 - USB 3.0 Host 控制器支持除 SuperSpeedPlus 之外的所有速度 - USB 2.0 Host 控制器支持低速，全速和高速

充当 USB Host 的嵌入式系统在部分程度上取决于系统支持的设备，因此支持的速度可能较低。

总线速度决定了信息在总线上传播的速率。除应用程序数据外，总线还必须携带状态，控制和错误检查信息，并且多个设备可以共享总线。因此，单个设备的应用程序数据的吞吐量略低于总线速度。

USB 1.0 规范定义了低速和全速。 全速适用于使用 RS-232（串行）和并行端口的大多数外围设备。 全速数据传输速率可与这些早期接口的速度相媲美。 因为电缆要求不那么严格，所以电缆更灵活，因此我建议使用低速。 低速设备的部分铜缆可能会降低制造成本。 随着 USB 2.0，USB 3.0 定义的 SuperSpeed 和 USB 3.1 定义的 SuperSpeedPlus 的发布，高速成为一种选择。
